## ūüĆü Vision

Every week, incredible products are built by Indian makers ‚ÄĒ but they don't get the launchpad they deserve. **BharatHunt** changes that.

A place where:
- **Makers** ship their products and get real feedback from a community that cares
- **Early adopters** discover the next big thing before everyone else
- **The Indian tech ecosystem** gets the spotlight it has always deserved

No gatekeepers. No editorial review. Just pure, community-driven discovery ‚ÄĒ ranked weekly, reset every Friday.

> ūüáģūüá≥ *Bharat me bana, duniya ke liye.*

## ūüéĮ Features

### ūüĎ§ For Makers
- **Launch instantly** or **schedule** for the next weekly cycle
- Beautiful product pages with logo, hero image, gallery, and video embeds
- Real-time feedback via community votes and comments
- Maker profiles to showcase everything you've built

### ūüó≥ÔłŹ For the Community
- **Weekly ranked feed** (Friday ‚Üí Thursday) ‚ÄĒ every product gets a fair shot
- One upvote per product per user ‚ÄĒ your vote matters
- Category filters to find exactly what interests you
- Nested comments for genuine conversations

### ūüĒí Built Right
- **All auth handled server-side** ‚ÄĒ JWTs in memory, refresh tokens in HTTP-only cookies
- **Rate limited, validated, sanitized** ‚ÄĒ bots and bad actors kept out
- **Ownership enforced everywhere** ‚ÄĒ you own your content, period

## ūüŹó Architecture

```
bh/
‚Ēú‚ĒÄ‚ĒÄ packages/
‚Ēā   ‚Ēú‚ĒÄ‚ĒÄ frontend/          # Next.js + Tailwind CSS app
‚Ēā   ‚Ēú‚ĒÄ‚ĒÄ backend/           # Express + TypeScript API
‚Ēā   ‚ĒĒ‚ĒÄ‚ĒÄ shared/            # Shared types, constants, contracts
‚Ēú‚ĒÄ‚ĒÄ package.json           # Turborepo root
‚ĒĒ‚ĒÄ‚ĒÄ turbo.json             # Pipeline configuration
```

**Architecture rules:**
- `frontend ‚Üí shared ‚Üź backend`
- Backend follows: `routes ‚Üí controllers ‚Üí services ‚Üí db`
- Services are pure TypeScript ‚ÄĒ zero HTTP knowledge
- All contracts live in `packages/shared` ‚ÄĒ frontend and backend stay in sync

Read the full design decisions in [ARCHITECTURE.md](ARCHITECTURE.md).
